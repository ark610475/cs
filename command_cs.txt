git init                      # 初始化 Git 儲存庫(建立.git)
git status                    # 查看狀態 (U未追蹤 T已追蹤 S已暫存 C已提交) 
git add <file name>           # 加入file到暫存區
git add .                     # 加入所有檔案到暫存區
git commit -m "版本訊息"       # 提交紀錄

git config --global user.name "作者姓名"       #設定作者姓名
git config --global user.email "作者電子郵件"   #設定作者電郵
git log     #版本紀錄 按q退出

git diff  <版本號> --<檔案名>      #查看該檔案目前內容與版本號內容差異
git checkout <版本號> --<檔案名>   #確定將檔案改回該版本(需再次commit)
git reset --hard <版本號>         #所有檔案還原，刪除之後版本

目錄建立".gitignore" 內容寫入需要忽略的檔名 可以避免被追縱

#github
git remote add origin https://github.com/ark610475/cs.git  #在Repo中建立
git branch -M main
git push -u origin main
#之後做完想更新Repo一定要使用 git push

#協作
 到Repo-> setting -> Collaborators -> add

 到信箱收取協作信件
 成為協作者之後 使用 git clone <Repo URL>來複製檔案到終端
 git pull      # 拉取遠端更新
 git log       # 查看提交紀錄
 git branch    # 目前分支
 git checkout -b <branchname>    #建立新分支
 git push origin <branchname>    #推送至某分支
 git fetch                       #分支表

 更新後可以到Github 的 branches 中找到自己的分支 選單中選取pull request
 其他人可以看到pull request並審核更新的部分
 發現可改進->評論區中留言  或右上角請求修改
 沒問題->merge  與主分支合併
